Return objects

User

 codeforcesのユーザーの情報が返されます。
パラメータ 説明
handle str型 codeforcesのハンドルネーム
email str型 連絡を取るためのメールアドレス(但しユーザーが公開設定で登録している場合)
vkId str型 VK(主にロシア系？のSNS)のユーザーID(但しユーザーが公開設定で登録している場合)
openId str型 OpenIDのユーザーID(但しユーザーが公開設定で登録している場合)
firstName str型 人名。海外圏ではファーストネーム。日本語圏では下の名前。空欄の場合がある。
lastName str型 人名。海外圏ではラストネームかファミリーネーム。日本語圏では上の名前。空欄の場合がある。
country str型 国。現在の居住地か出生地かは規定なし？ 空欄の場合がある。
city str型 都市。現在の居住地か出生地かは規定なし？ 空欄の場合がある。
organization str型 所属組織。学校名や勤務先など。空欄の場合がある。
contribution int型 contribution、詳細不明。blogでいいねをされるか、friend登録をされると増える？
rank str型 現在のレーティングに応じた呼称(SpecialistやExpertなど)。
rating int型 現在のレーティング。
rank str型 レーティングに応じた呼称(SpecialistやExpertなど)。
maxRank str型 現在までの成績で最も成績の良かった呼称。
maxRating str型 現在までの成績で最も高いレーティング。
lastOnlineTimeSeconds int型 最新のオンラインの時間(UNIX時間表記)。codeforcesのユーザーページではonline nowと表記がされる場合もAPIで得られるのはUNIX時間。
registrationTimeSeconds int型 codeforcesにユーザー登録した時間(UNIX時間表記)。
friendOfCount str型 friend登録されている人数。
avatar str型 ユーザーアイコン(小)のURL。
titlePhoto str型 ユーザーアイコン(大)のURL。


BlogEntry

 codeforcesのblog記事が返されます。
パラメータ 説明
id int型 該当のblogの記事に割り当てられたID
originalLocale str型 該当のblogの記事が書かれた言語。en(英語)やru(ロシア語)の2種類あることを確認。
creationTimeSeconds int型 該当のblogの記事が書かれた時間(UNIX時間表記)。
authorHandle str型 該当のblogの記事を書いたユーザーのハンドルネーム
title str型 該当のblogの記事の件名
content str型 該当のblogの記事の本文
locale str型
modificationTimeSeconds int型 該当のblogの記事が更新されたUNIX時間
allowViewHistory bool型 trueならば該当のblogの記事の変更履歴が見られる？
tags str型 該当のblogの記事に設定されているタグ
rating int型 該当のblogの記事の評価

Comment

 codeforcesのblog記事のコメントが返されます。
パラメータ 説明
id int型 おそらくは投稿コメントに割り当てられたID。
creationTimeSeconds int型 コメントの投稿日時(UNIX時間表記)。

locale str型 投稿の言語。en(英語)やru(ロシア語)。
text str型 コメント本文。
parentCommentId int型 親コメントのID。既存のコメントへの返信でなければおそらく空欄になる。
rating int型 いいねの数。
